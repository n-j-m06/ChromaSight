# 🎨 ChromaSight
**AI-Powered Color Identification & Computer Vision Tool**

ChromaSight is an intelligent color recognition system that leverages Computer Vision and Machine Learning to identify colors from images and real-time video streams with high accuracy. This project explores the intersection of image processing and classification algorithms to solve practical problems in accessibility and design.

---

## 🚀 Key Features
*   **Real-time Color Detection:** Instant identification of colors through webcam or static image input using OpenCV.
*   **KNN-Based Classification:** Utilizes the K-Nearest Neighbors (KNN) algorithm for precise color mapping and naming.
*   **Detailed Analytics:** Provides RGB and Hex code values for identified shades.
*   **Research-Oriented:** Developed alongside a 12-page technical research paper detailing methodology and performance metrics.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Libraries:** OpenCV, NumPy, Pandas
*   **Algorithm:** K-Nearest Neighbors (KNN)
*   **Dataset:** Custom-mapped color CSV datasets for expanded recognition.
